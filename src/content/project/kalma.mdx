## Overview
Kalma is a mental health application which focuses on the user's self healing so users will not do a self diagnose that will lead to 
something bad. The primary goal of this app is to make users do the self healing method for their daily concern of their mental health 
state. This app provides 3 main features.

## Solution & Approach
The mobile app was developed with <strong>Flutter</strong>. It is interesting because my team and I had to create a robust mobile 
architecture, which was challenging since it was our first real-life mobile app project.

To improve app performance and make it dynamic for users, we used several key libraries:
